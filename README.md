# 📐 NumPy Learning

A collection of Jupyter notebooks for learning and practicing **NumPy** — the fundamental Python library for numerical computing.

## 📂 Notebooks

| Notebook | Description |
|---|---|
| [`numpy_arrays.ipynb`](numpy_arrays.ipynb) | Creating arrays from lists, `arange`, zeros/ones, and random data generation |
| [`array_attributes.ipynb`](array_attributes.ipynb) | Exploring array properties: shape, ndim, dtype, size, and more |
| [`array_indexing_slicing.ipynb`](array_indexing_slicing.ipynb) | Indexing and slicing 1D, 2D, and multi-dimensional arrays |
| [`array_methods.ipynb`](array_methods.ipynb) | Common array methods: `sum`, `mean`, `min`, `max`, `reshape`, etc. |
| [`array_operations.ipynb`](array_operations.ipynb) | Element-wise arithmetic, broadcasting, and mathematical operations |
| [`array_stacking_splitting.ipynb`](array_stacking_splitting.ipynb) | Stacking arrays (`hstack`, `vstack`) and splitting them |
| [`boolean_indexing.ipynb`](boolean_indexing.ipynb) | Filtering arrays using boolean conditions |
| [`numpy_exercise.ipynb`](numpy_exercise.ipynb) | Hands-on exercises using a student marks dataset (aggregations, filtering, broadcasting) |
| [`sudoku_valid.ipynb`](sudoku_valid.ipynb) | NumPy-powered Sudoku board validator (rows, columns, and 3×3 sub-grids) |

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook or JupyterLab

### Installation

```bash
# Clone the repository
git clone https://github.com/m-alihassan-dev/numpy-learning.git
cd numpy-learning

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter
jupyter notebook
```

## 🧰 Tech Stack
